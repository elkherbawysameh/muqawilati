# مقاولاتي - تطبيق إدارة مشاريع التشطيبات

تطبيق ويب متكامل لإدارة مشاريع التشطيبات والمقاولات.

## المتطلبات
- Node.js (الإصدار 16 أو أحدث): https://nodejs.org

## تشغيل التطبيق
1. افتح مجلد `muqawilati` في terminal أو Command Prompt
2. نفّذ:
   ```
   npm install
   node server.js
   ```
3. افتح المتصفح على: http://localhost:3000

## المميزات
- إدارة العملاء: بيانات - بنود العقد - دفعات - صور - تقدم الإنجاز
- إرسال تذكير للعميل عبر واتساب بضغطة زر
- إدارة العمال والصناعية مع تصنيفات قابلة للتعديل
- إصدار وطباعة فواتير مع حساب الضريبة
- البحث عن أسعار الخامات عبر جوجل
- نمط ليلي / نهاري
- قاعدة بيانات محلية (لا تحتاج إنترنت)

## البيانات
تُحفظ تلقائياً في ملف `muqawilati.db` داخل نفس المجلد.
